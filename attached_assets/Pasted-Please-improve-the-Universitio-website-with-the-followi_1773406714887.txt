PART 2 — FREE CONSULTATION FORM PAGE

Create or update the Free Consultation form page to be extremely user-friendly, clean, and conversion-focused.

Top intro text:
At the top of the form, include a short helpful explanation telling users that:
- the more accurately they complete the form
- the better we can understand their profile
- and the better we can guide them

Use polished, supportive British English.

The form should feel modern, structured, and easy to complete.
Use clear grouped sections, spacing, progress-friendly layout, and excellent mobile UX.

Important:
Some fields are required, and some have conditional logic.

FORM FIELDS AND LOGIC

1. Full Name
- Required

2. Mobile Number
- Required
- Must include country code selector/dropdown
- Should feel modern and international-user friendly

3. Email Address
- Required

4. Date of Birth
- Required

5. Previous Education
Ask the user to enter:
- field of study previously studied
- level of study previously completed

The user must be able to add more than one previous qualification/education entry if needed.
For example:
- Bachelor’s in Business
- Diploma in IT
Use a repeatable input/group system.

6. Intended Course Area
Ask what subject/course area they want to study.
- Required
- Include a list of major course areas
- Also include “Other”
- If “Other” is selected, the user must type their intended field manually

Include a good list of common course areas such as:
- Business and Management
- Computer Science / IT
- Engineering
- Health and Nursing
- Medicine
- Law
- Architecture
- Art and Design
- Education
- Social Sciences
- Psychology
- Media and Communications
- Hospitality and Tourism
- Finance and Accounting
- Marketing
- Data Science / AI
- English / Humanities
- Other

7. Nationality
- Required
- Use a complete country list dropdown

8. Preferred Study Destination
- Required
- User can select maximum 2 countries only
- Use these options:
  - United Kingdom
  - United States
  - Canada
  - Germany
  - Netherlands
  - Australia

Make this clear in the UI:
- “You may choose up to 2 destinations”

9. English Language Qualification
- Required
Ask:
Do you have an English language qualification?

Options:
- Yes
- No

If Yes:
Ask:
- Which qualification do you have?
Use options such as:
  - IELTS Academic
  - TOEFL
  - PTE Academic
  - Duolingo English Test
  - Cambridge English
  - Other

If “Other” is selected, ask them to specify.
Also ask:
- Overall score
Only ask for overall score, not sub-scores.

If No:
Ask:
- What is your current English level?
Required
Use selectable options such as:
  - Beginner
  - Elementary
  - Pre-Intermediate
  - Intermediate
  - Upper-Intermediate
  - Advanced

10. Tuition Budget
- Required

Add a short helper note making it clear that users should provide realistic information so that we can guide them properly and recommend suitable options.

Budget options:
- Under $10,000
- Between $10,000 and $20,000
- Over $20,000

If user selects “Under $10,000” show an important informational note:
- for study in the UK, this budget is generally not sufficient
- for Canada and the USA, options may be limited
- for some European destinations, there may still be possible pathways

This should be presented as a helpful advisory note, not an error.

11. CV

Options:
- Yes
- No

12. Marital Status
- Required
Ask:
Are you single or married?

This information helps us understand suitability for certain study pathways, especially because some countries, such as the UK, have restrictions around dependant visas in some study categories.

Options:
- Single
- Married

Conditional logic:
If user selects:
- Married
AND

For study in the UK, if you need a dependant visa route, you may need to apply through eligible research-based programmes, which typically require a research proposal.

This should be informative and clearly visible.
