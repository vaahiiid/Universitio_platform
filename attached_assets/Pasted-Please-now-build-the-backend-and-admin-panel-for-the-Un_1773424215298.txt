J. BLOG IMPORT DATA FORMAT
==================================================

The expected blog data file inside the ZIP should be JSON.

Design the import logic around a posts.json structure such as:

{
  "posts": [
    {
      "title": "Post title",
      "slug": "post-slug",
      "date": "2026-03-10",
      "excerpt": "Short summary",
      "content": "<p>Full article content...</p>",
      "categories": ["Category 1", "Category 2"],
      "tags": ["tag1", "tag2"],
      "featured_image": "example-image.jpg",
      "status": "published"
    }
  ]
}

For now, build the system to support this structure.
The UI should explain that:
- the ZIP should contain posts.json
- images must be inside an images folder
- featured_image values should match image file names
